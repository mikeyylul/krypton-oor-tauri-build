KRYPTON SOLUTIONS OOR - TAURI OFFLINE WINDOWS APPLICATION

Recommended option
1. Run the included Krypton Solutions OOR installer.
2. Open Krypton Solutions OOR from the Start menu.

Portable option
1. Open the Portable folder.
2. Double-click KryptonOOR.exe.

LOCAL AND OFFLINE DATA
- The application does not use a cloud database or online storage.
- Jobs, quotes, shortages, actions, spreadsheet imports, photo scans, and autosaves stay on this computer.
- Saved information restores automatically when the application is reopened.
- The application has no automatic online updater.
- The installer includes the WebView2 runtime needed for offline installation.

PHOTO SCANNING
- OCR worker, engine, and English recognition data are included inside the application.
- New-job screenshots, RFQ photos, old job bookings, and shortage-list images scan without downloading recognition files.

BUILD
- This Windows x64 package is compiled and validated by the repository's Windows build workflow.

WINDOWS NOTICE
This application is not code-signed. Windows may display an Unknown Publisher warning. Select More info, then Run anyway only if you downloaded this package directly from its trusted source.
